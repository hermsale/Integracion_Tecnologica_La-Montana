Tener instalado IDE Arduino

Tener instalado la extension de wokwi

- se corre el diagram.json y se da play
- si hay alguna modificación a realizar al proyecto, en lo que respecta código, se debe hacer en el IDE, y despues pulsar sketch -> export compile binary
- si hay que realizar modificaciones a nivel hardware, se debe modificar desde el entorno online de wokwi
- Si se modifico el proyecto desde wokwi online, y se quiere correr local. Se debe descargar el zip del proyecto
    - descomprimirlo en la carpeta raiz donde esta el wokwi.toml. 
    - Abrir el IDE, importar el proyecto. y repetir el paso de exportar el compilado binario

Enlace publico: https://wokwi.com/projects/467855455079230465
